# FindNumber

### Application
The game to find number

### Functionality
Created a settings menu in which you can set the game time.
It is possible to play against the clock and without it.
Created a section record. Data is stored in UserDefaults.

### Stack
+ Swift.
+ UIKit.
+ Storyboards.
+ Auto Layout.
+ Codable.
+ UserDefaults.
+ Animate.
+ Alerts.
+ MVC.
